A simple way to change a post's type in WordPress

== Description ==

This plugin adds a post-type drop-down to the post editor section, you can change any post to a new post type.

= Features =
* easy to install
* No need to settings in admin.
* Change Page to Post
* Change  Post to Page
* Change Post to Custom
* Change Custom post to Custom post


= Requirements =
This plugin requires PHP 5 or above and WordPress 3.0 or greater. It works in standard WordPress environments. It is strongly recommended to use WordPress in the newest version.

== Installation ==

1. Go to Plugins > Add New or Download the plugin and unzip it. 
2. Searh "Post type manager" Or Upload the folder to your /wp-content/plugins/ folder.
3. install and activate the WordPress admin panel.


== How to Use ==

1. From the post edit screen, above the “Publish” button is the “Post Type” interface.
2. From the post list screen, click quick edit below the “status” button select post type and update post.
